有的人会误认为内存要开成 $N\log V$，其中 $V$ 为 $10^9$，而大多数情况下存一个直线是需要至少一个 int 一个 long long 的，如果 $N$ 是 $10^6$ 再乘个 30 就很容易炸。实际上这是一种误解。

仔细观察实现过程，这个线段树没有分割后分别递归，而是只会往一侧走。

每次存下直线后都会直接停止，从而可以知道每次 add 操作只会新建一个结点，所以内存开成 $N$ 即可。
